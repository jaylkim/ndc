# ndc

<!-- badges: start -->
<!-- badges: end -->

This package includes a function to convert NDCs in 10-digits to
11-digits.

## Installation

``` r
remotes::install_github("jaylkim/ndc")
```

## Example

``` r
library(ndc)
## basic example code
ndc_convert("path/to/file.csv")
```
